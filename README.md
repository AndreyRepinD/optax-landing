# OptAx — лендинг

Одностраничный сайт движка оптимизационного планирования OptAx: оптимизация цепочки поставок, мультиэшелонная оптимизация запасов, оптимизация загрузки транспортного средства, оптимизация производственного расписания.

- `index.html` — весь сайт одним файлом (HTML + CSS + JS), без сборки и зависимостей; шрифты подгружаются с Google Fonts.

## Публикация через GitHub Pages

Settings → Pages → Source: **Deploy from a branch** → Branch: **main** / **(root)** → Save.

Через 1–2 минуты сайт будет доступен по адресу: https://andreyrepind.github.io/optax-landing/
